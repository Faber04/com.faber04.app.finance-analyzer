# 📝 Note per l'Agent

**Come usare questo template:**

1. **All'inizio sessione**: Copia questo template, rinomina con numero sessione
2. **Durante sessione**: Compila progressivamente mentre lavori
3. **Fine sessione**: Completa tutte le sezioni
4. **Salva in**: `/docs/sessions/session-[N].md`

**Non saltare sezioni**, anche se brevi. La documentazione completa è essenziale per la continuità.
